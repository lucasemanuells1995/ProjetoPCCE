# ProjetoPCCE / Django / Python / HTML

Projeto semi-completo, de um almoxarifado virtual para a PCCE.
LÓGICA FEITA.
CÓDIGO COMENTADO.
'127.0.0.1:8000/admin' FUNCIONANDO (usarname: admin   senha: admin)
BASE DE DADOS SQLITE.

ÚNICO PROBLEMA É CONSEGUIR DIFERENCIAR ARMA DE MUNIÇÃO,
ENTRETANTO, ELE MOSTRA OS DOIS OBJETOS RELACIONADOS AO CALIBRE CORRETO.

- Abrir o CMD ou PowerShell e dar 'pip install django'
- Executar o comando 'python manage.py runserver'
- Abrir o link '127.0.0.1:8000/almoxarifado/' no navegador (com o servidor local ligado).
- Testar todas as opções que aparecerem no site.
